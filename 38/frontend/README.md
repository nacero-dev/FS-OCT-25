# Blog App

Aplicación de blog full stack donde los usuarios pueden registrarse, iniciar sesión y crear publicaciones con imágenes obligatorias en Base64.  
Los usuarios no autenticados solo pueden leer publicaciones y comentarios.

Tecnologías: Node.js, Express, MongoDB, JWT, React, Context API, Vite.

Frontend: https://38-front.vercel.app  
Backend: https://38-backend.vercel.app

Instalación: configurar variables de entorno y ejecutar `npm install` + `npm run dev`.
